# < 커머스 과제 >

## 목적

여러 객체가 협력하는 객체 지향 설계 구조를 커머스 플랫폼에 적용해보는 경험을 쌓기.

상품 등록과 출력, 카테고리별 관리, 고객 주문과 재고 차감, 장바구니 기능 등 기본적인 커머스 플랫폼의 핵심 기능 구현

### [필수 기능]
 - [x]  STEP 1. 객체 지향 설계를 적용해 상품 관리 시스템을 프로그래밍
 - [x] STEP 2. 객체 지향 설계를 적용해 순서 제어를 클래스로 관리하기
 - [ ] STEP 3. 객체 지향 설계를 적용해 상품 카테고리와 고객 관리를 클래스 기반으로 관리하기
 - [ ] STEP 4. 캡슐화 적용
